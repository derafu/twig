Set de pruebas

Se envían al SII los DTE estándares que ellos piden validar del set solicitado.

Simulación

Se envían al SII ejemplos de DTE reales que la empresa emitirá en producción.

Intercambio

Se envían al SII los XML asociados a la respuesta a un intercambio de DTE.

Muestras de PDF

Se envían al SII los PDF de todos los DTE del set de pruebas y simulación.
